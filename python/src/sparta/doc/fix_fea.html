fix fea command

Syntax: 

fix ID fea ELMER_EXE ELMER_SIF_FILE

  • ID is documented in fix
  • command = style name of this fix command
  • ELMER_EXE = path to elmer executable
  • ELMER_SIF_FILE = path to input file for elmer

    • Examples: 

    fix 1 field/grid gradBx gradBy NULL
