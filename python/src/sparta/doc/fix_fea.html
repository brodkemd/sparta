fix fea command

Syntax: 

fix ID fea ELMER_EXE ELMER_SIF_FILE

  • id = documented in fix
    • , just in set to something
  • style = style name of this fix command, for this it is fea
  • nevery = execute every this many time steps
  • elmer_exe = path to elmer executable
  • elmer_sif_file = path to the sif file to run with elmer
  • surf_file = path or name of file to dump surface to
  • compute_id = id for compute
  • group_id = group ID for which surface elements to perform calculation on
  • mix_id = mixture ID for particles to perform calculation on
  • dump_id = id for the surf dump command for the data need for elmer
  • dump_file = file for the surf dump command for the data need for elmer
  • select_id = what surface elements to dump
  • dump_compute_format = this is has to be c_[*]

    • Examples: 

    fix 1 fea 2 ELMEREXE case.sif out.surf 5 all all 8 out.dump all c_5[*]

    See Syntax.
